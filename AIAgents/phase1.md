# Install Python and VSCode
1. install python.
1. install vs code.
1. vs code extensions:
   - Python (by Microsoft)
   - Black Formatter (by Microsoft)
   - Jupyter (by Microsoft)
1. The Critical Setup Step (Don't Skip)
The most common error beginners face is VS Code not "knowing" where your Python is installed.

    1. Open VS Code.
    1. Press Command + Shift + P (Command Palette).
    1. Type: Python: Select Interpreter and click it.
    1. Select the version that says "Recommended" (e.g., Python 3.x.x).
    1. If nothing shows up: You may need to restart VS Code or ensure Python installed correctly.
