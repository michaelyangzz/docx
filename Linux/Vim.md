# How to use Vim in Linux Mac
## Two modes in Vim
1. Command Mode  
1. Insert Mode
## Open a file
- vi/vim test.txt
## Modes
1. **ESC**: Command Mode
1. **i**: Insert Mode
1. **I**: Insert text at the begining of current line
## Select
1. **v**: Select text
1. **V**: Select line
## Undo
- **u**: Undo
## Copy and Paste
1. **y**: Copy selection
1. **p**: Paste
## Save
1. **:w**: Save
1. **:wq**: Save and Quit 
1. **:q!**: Force quit without saving
